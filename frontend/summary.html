Endless Legend 2 Game Summary Viewer

Endless Workshop can load Endless Legend 2 end-game report exports so players can inspect a run’s high-level summary, technology order, empire stats, and city details.

What the summary page covers

  • Upload a local end-game report JSON or preview the example report.
  • Review overview, tech progress, empire statistics, and city breakdown tabs.
  • Use reports exported by the End Game Report mod after reaching the victory screen.

Interactive report loading

The live viewer reads report files in the browser after the page starts, so report analysis works without requiring a database-backed server render.
