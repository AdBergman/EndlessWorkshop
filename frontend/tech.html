Endless Legend 2 Tech Tree Planner

Endless Workshop includes an interactive Endless Legend 2 tech tree planner that helps players inspect research unlocks, compare eras, and share build ideas.

What the planner covers

  • Browse the technology tree by era and view unlock relationships.
  • Track selected research paths in a companion spreadsheet view.
  • Use shared-build links or imported progress to review a specific tech route.

Interactive data loading

The live planner loads the latest available game data after the app starts, so technology names, unlocks, and build state stay aligned with the current workshop dataset.
