About Endless Workshop

Endless Workshop is a fan-made Endless Legend 2 project focused on practical planning, reference, and post-game analysis tools for the community.

Current tools

  • An interactive tech tree planner for research-path exploration.
  • A unit evolution explorer for faction rosters and progression paths.
  • A report viewer for end-game summaries, tech progress, empire stats, and city breakdowns.

Project direction

The project continues to expand with codex reference material and mod support while staying aligned with the game as its systems evolve.
