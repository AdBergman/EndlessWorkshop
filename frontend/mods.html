Endless Legend 2 Mods and Tools

Endless Workshop curates the EL2 Essentials Pack plus supporting utility mods, with installation notes and direct release links for players who want a cleaner Endless Legend 2 experience.

Featured content

  • The EL2 Essentials Pack bundles WorldGen, BulkTrade, and SyntheticHarmony.
  • Support tools include Quest Recovery and End Game Report.
  • The page also explains the BepInEx requirement and a short installation flow.

Installation overview

Install BepInEx 5.x for Windows x64, launch the game once, then extract the chosen mod package into the Endless Legend 2 folder so the plugin files land under BepInEx.
