## Purpose
Strategy View is the planning and decision surface.

## Core Interaction Model

Inspect current chapter
-> compare branch options
-> simulate a path
-> inspect projected outcomes

## Main Sections
- Tactical Brief
- Current Objective
- Requirements
- Rewards
- Selected Path
- Decision Options
- Projected Outcome
- Path Markers

## Important
Strategy is not a graph canvas and not a duplicate Lore reader.
