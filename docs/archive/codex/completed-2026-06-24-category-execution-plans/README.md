# Completed Codex Category Execution Plans

Archived: 2026-06-24

This folder contains durable execution plans from completed Codex category
evolution runs. They were moved out of `docs/active/` during
`DOCS-AUDIT-002` to reduce active documentation noise.

Use the active category evolution docs first when reopening a category. These
plans are historical task-state records and should be read only when the
execution history of a completed category pass matters.
