# EWShop Quest Explorer — Lore View Design Document

## Purpose
Lore View is the narrative chronicle experience.

## Core Interaction Model

Step 1 lore
-> choice
-> selected Step 2 lore / outcome
-> choice
-> selected Step 3 lore / outcome
-> chapter exit / next chapter

Future lore must not appear before the choice that reveals it.

## Continuous Lore Reader
Long-term Lore View should support:
Chapter 1 -> selected path -> Chapter 2 -> selected path -> Chapter 3

The stream stops naturally at unresolved choices.
